# SBCompletedAuction

The `SBCompletedAuction` class extends the [`SBAuctionBase`](object-model/sbauctionbase) class and represents an ended
auction.

| **Property Name** | **Type** | **Description**                             |
|-------------------|----------|---------------------------------------------|
| `winner`          | `Bool`   | Indicates whether the user won the auction. |
